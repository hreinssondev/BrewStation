BrewStation!

Control and view all your Homebrew apps in one place.
Save you apps and all of their settings and configs with a snapshot and restore them with one click on another Mac or the same Mac

Can be used as a Homebrew overview and manager.

Can be used as a Restore manager that backs up and restores your apps (full snapshot).

Can be used as a New Mac setup tool, save apps from one Mac and install and restore them (Fully) with one click on another Mac.
